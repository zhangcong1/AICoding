# 文档目录

本目录存放**需长期沉淀、人类可读**的文档。规范与需求真理源在 `spec/`，不在此处。

## 子目录

| 目录 | 放什么 | 示例 |
|------|--------|------|
| `guides/` | 团队培训、实践指南、操作手册 | AICoding 开发实践 |
| `architecture/` | 架构决策记录（ADR）、技术说明 | ADR-001-选用 PostgreSQL |
| `user/` | 用户向说明、上线公告、FAQ | 任务管理系统使用指南 |

## 与 spec/ 的分工

| 问题 | 答案 |
|------|------|
| 当前系统**必须**做什么？ | `spec/specs/` |
| 这次迭代**要改**什么？ | `spec/changes/{name}/` |
| 团队怎么协作、怎么培训？ | `docs/guides/` |
| AI 草稿还没确认？ | `spec/AI2AI/` |
| 人类原始需求输入？ | `spec/ME2AI/` |

## 命名建议

- 指南：`{主题}-guide.md`
- ADR：`ADR-{序号}-{简短标题}.md`
- 用户文档：按功能模块分子目录
