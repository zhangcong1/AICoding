# Personal Workbench

个人工作台：基于 [AICoding](https://github.com/zhangcong1/AICoding) 的独立开发分支，用于个人效率工具、实验与日常沉淀。

## 分支说明

| 分支 | 用途 |
|------|------|
| `main` | 上游主仓库基线 |
| `personal-workbench` | 个人工作台专属迭代 |

## 目录结构

| 路径 | 说明 |
|------|------|
| `spec/` | 需求与规范真理源（`specs/`、`changes/`、`ME2AI/`、`AI2AI/`） |
| `docs/` | 长期沉淀的人类可读文档 |
| `src/` | 应用与脚本源码 |
| `scripts/` | 自动化与辅助脚本 |
| `相关文档/` | 业务相关材料 |

更多文档组织方式见 [docs/README.md](docs/README.md)。

## 快速开始

```bash
git clone -b personal-workbench https://github.com/zhangcong1/AICoding.git
cd AICoding
cp .env.example .env   # 按需配置环境变量
```
